# RemindMi
## Overview
Introducing RemindMi, a user-friendly task management mobile app for iOS and Android that empowers individuals to efficiently organize daily activities, deadlines, and to-do lists with ease and convenience. The app is especially designed with accessibility in mind. A user can create one or many dependent accounts which they can manage, create tasks for, and monitor the progress of the dependent users.

With the aim of ensuring that the app caters to all users, it has been designed with accessibility in mind. This means that individuals with disabilities, whether visual or auditory, can easily navigate through the interface and utilize its features.

One of the unique features that sets this task management app apart from other similar apps is the ability for a user to create and manage dependent accounts. This means that a user can create one or many accounts for individuals who are dependent on them, such as children or elderly family members, and then manage these accounts through the app. By creating these dependent accounts, a user can easily assign tasks and deadlines to each account and monitor the progress of the dependent users in real-time.
